Issuing Elements allows you to display the sensitive data of your Issuing cards in a PCI-compliant manner.

API Reference for Issuing Components

import {
  StripeIssuingCardNumberDisplayComponent,
  StripeIssuingCardExpiryDisplayComponent,
  StripeIssuingCardCvcDisplayComponent,
  StripeIssuingCardPinDisplayComponent
} from 'ngx-stripe'

Components

StripeIssuingCardNumberDisplayComponent

Selector: ngx-stripe-issuing-card-number-display

Properties

Name Description
@Input()
containerClass: string 		A CSS class to add to the container element.
@Input()
options: StripeIssuingCardNumberDisplayElementOptions 		An object containing the configuration options for the Stripe Issuing Card Number Element. See the Stripe documentation for further details.
@Input()
elementsOptions: StripeElementsOptions 		DEPRECATED: USE ELEMENTS INSTEAD
An object containing the configuration options for the Stripe Elements instance.
@Input()
stripe: StripeServiceInterface 		DEPRECATED: USE ELEMENTS INSTEAD
The StripeServiceInterface instance to use when creating the Stripe Elements instance.
@Output()
load: EventEmitter<
StripeIssuingCardNumberDisplayElement
> 		CUSTOM: NGX-STRIPE ONLY It is triggered when the Angular Component is ready to be used. It emits the Stripe Issuing Card Number Web Element instance.
element: StripeIssuingCardNumberDisplayElement CUSTOM: NGX-STRIPE ONLY The Stripe Element

Methods

update(options: StripeIssuingCardNumberDisplayElementOptions)
Updates the options the Element was initialized with. Updates are merged into the existing configuration.
